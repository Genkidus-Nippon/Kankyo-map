# 環境世界地図

世界地図から各国の環境ニュースにアクセスできるツールです。

## かんたん起動（ターミナルにコマンドを打たない方法）

前提: パソコンに Node.js 18 以上が入っていること（未導入なら https://nodejs.org からLTS版を入れてください）。

- **Windows**: `start-windows.bat` をダブルクリック
- **Mac**: `start-mac.command` をダブルクリック
  - 初回だけ「開けません」と出たら、ファイルを右クリック →「開く」→「開く」を選ぶと許可できます。

初回は自動でセットアップ（`npm install`）が走り、その後ブラウザで自動的に
**http://localhost:3000** が開きます。以降はダブルクリックだけでOKです。

> 使用中は黒い画面（ターミナル）が1つ開いたままになります。これはサーバーが動いている印で、
> 閉じるとサイトも止まります。サイトを使い終わったらその画面を閉じてください。

### ターミナルから起動する場合（従来どおり）
```bash
npm install   # 初回のみ
npm start
```

### 起動できているかの確認
ブラウザで `http://localhost:3000/api/health` を開き、`{"ok":true,"fetch":true}` が出ればOK。
`"fetch":false` の場合は Node.js が古いので18以上に更新してください（`node -v` で確認）。

## メール送信（問い合わせ）の設定 — Gmailの場合

`.env.example` を `.env` にコピーし、次のように埋めます。

| 項目 | 入れる値 |
|------|----------|
| SMTP_HOST | `smtp.gmail.com` |
| SMTP_PORT | `587` |
| SMTP_USER | 自分のGmailアドレス（例: `kihahiro5463@gmail.com`） |
| SMTP_PASS | **アプリパスワード**（下記の手順で作る16桁） |
| MAIL_TO | 受け取りたいアドレス（自分のGmailでOK） |
| MAIL_FROM | 自分のGmailアドレス（SMTP_USER と同じ） |

**アプリパスワードの作り方（重要）**
Gmailは通常のログインパスワードでは送信できません。次の手順で専用パスワードを作ります。
1. Googleアカウントで「2段階認証」を有効にする（有効でないとアプリパスワードは作れません）。
2. https://myaccount.google.com/apppasswords を開く。
3. 名前を適当に（例: env-world-map）入力して作成。
4. 表示された16桁（例: `abcd efgh ijkl mnop`）を **スペースを詰めて** `SMTP_PASS` に貼る。

設定後、サーバーを再起動すると起動ログに「メール送信: 有効 (SMTP)」と出ます。
`.env` を設定しない場合でも、問い合わせ内容は `submissions.json` に保存されます。

## 「たまにしかニュースが出ない」とき
- `GNEWS_KEY` 未設定だと無料の GDELT を使いますが、短時間に連続アクセスすると制限され空振りしやすいです。
- **GNEWS_KEY を設定すると安定します**（日本語・1日100件まで）。
- サーバーのターミナルに `GNews失敗: 403` と出たらキーが不正、`429` なら当日の上限です。

## ずっと立ち上げっぱなしにしたい / ターミナルを一切見たくない
無料のホスティング（Render, Railway など）にこのフォルダを配置すると、常時稼働してURLだけでアクセスできます。
必要なら手順をご案内します。
